# HR_Analytics_PowerBI_Project
Interactive HR Analytics dashboard built in Power BI to analyze employee attrition and workforce trends.
This project showcases an interactive HR Analytics dashboard created in Power BI to explore employee attrition patterns and overall workforce trends. The goal was to take raw HR data and turn it into meaningful insights that can help HR teams and business leaders make better decisions.
